ðŸ”¹ Common Git Commands for Daily Use
| Command                       | Purpose                                 | Example                                        |
| ----------------------------- | --------------------------------------- | ---------------------------------------------- |
| `git status`                  | Check which files are modified / staged | `git status`                                   |
| `git add <file>`              | Stage specific file                     | `git add index.html`                           |
| `git add .`                   | Stage all changes                       | `git add .`                                    |
| `git commit -m "message"`     | Save changes locally                    | `git commit -m "Fixed bug"`                    |
| `git log`                     | See commit history                      | `git log`                                      |
| `git branch`                  | Show current branch                     | `git branch`                                   |
| `git checkout <branch>`       | Switch branch                           | `git checkout dev`                             |
| `git checkout -b <branch>`    | Create + switch branch                  | `git checkout -b feature1`                     |
| `git remote -v`               | Show remote repo URL                    | `git remote -v`                                |
| `git remote add origin <url>` | Connect GitHub repo                     | `git remote add origin https://github.com/...` |
| `git push`                    | Push changes to GitHub                  | `git push`                                     |
| `git push -u origin master`   | First-time push                         | `git push -u origin master`                    |
| `git pull`                    | Pull latest changes from GitHub         | `git pull origin master`                       |
| `git diff`                    | See changes not yet staged              | `git diff`                                     |

ðŸ”¹ Common Errors & Fixes
| Error                                   | Solution                                                 |
| --------------------------------------- | -------------------------------------------------------- |
| `fatal: remote origin already exists`   | Delete existing remote first: `git remote remove origin` |
| `error: failed to push some refs`       | Pull first: `git pull origin master --rebase`, then push |
| `permission denied (publickey)`         | Setup SSH key or use HTTPS URL for remote                |
| `nothing to commit, working tree clean` | No changes staged; check with `git status`               |
